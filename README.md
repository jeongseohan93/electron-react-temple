💻 로스트아크 데스크톱 앱 (Lost Ark Desktop App)
로스트아크 유저들을 위한 최적화된 데스크톱 애플리케이션입니다. 게임 플레이를 더욱 편리하게 만들어주는 다양한 기능을 데스크톱 환경에서 제공하여, 게임 밖에서도 로스트아크의 주요 정보를 놓치지 않도록 도와줍니다.

✨ 주요 기능 하이라이트
캐릭터 정보 조회: 나의 캐릭터와 장비, 각인, 스킬 정보를 한눈에 확인하세요.

경매장 & 거래소:

경매장: 원하는 아이템을 빠르게 검색하고 최저가를 확인하세요.

거래소: 주요 아이템의 시세를 주식 그래프 형태로 제공하여, 시세 변동을 실시간으로 파악할 수 있습니다.

대시보드:

공지/이벤트: 로스트아크 공식 공지사항 및 이벤트 정보를 놓치지 마세요.

모험섬: 오늘의 모험섬 정보를 바로 확인하세요.

즐겨찾기: 자주 확인하는 거래소 아이템을 등록하여 시세를 추적하세요.

스마트 알림: 즐겨찾햣기 아이템의 가격이 목표 금액에 도달하면 데스크톱 알림을 보내드립니다.

시작 프로그램 등록: 컴퓨터 부팅 시 앱을 자동으로 실행하여 편리하게 사용할 수 있습니다.

API 키 관리: 프로그램 내에서 안전하게 API 키를 입력하고 관리할 수 있습니다.

개인 설정: 테마, 알림 등 다양한 옵션을 나에게 맞게 설정하세요.

🛠 기술 스택
이 프로젝트는 빠르고 안정적인 사용자 경험을 제공하기 위해 다음과 같은 기술 스택으로 구성되었습니다.

카테고리	기술	설명
데스크톱	Electron	웹 기술로 크로스 플랫폼 데스크톱 애플리케이션을 만듭니다.
데이터베이스	SQLite	서버가 아닌 로컬 환경에서 데이터베이스를 사용합니다. 별도 설치 없이 빠르고 효율적인 데이터 관리가 가능합니다.
번들러	Vite	빠른 개발 환경과 최적화된 번들링을 제공합니다.
프런트엔드	React	유연하고 효율적인 사용자 인터페이스를 구축합니다.
스타일링	Tailwind CSS	유틸리티 우선 CSS로 빠르고 일관된 스타일링을 가능하게 합니다.
빌드/배포	Electron-builder	macOS, Windows, Linux 등 다양한 환경에 맞는 설치 파일을 만듭니다.


🌐 데이터 처리 방식
이 애플리케이션은 사용자 로컬 환경에서 로스트아크 공식 API 서버와 통신하여 필요한 데이터를 가져옵니다.

실시간 통신: API 키를 통해 공식 서버와 통신하여 캐릭터 정보, 경매장, 거래소 등의 최신 데이터를 가져옵니다.

데이터 가공 및 저장: 가져온 데이터를 SQLite 로컬 데이터베이스에 저장하고 가공하여 필요한 형태로 변환합니다.

결과 출력: 가공된 데이터를 그래프나 목록 등 보기 쉬운 형태로 사용자 인터페이스에 출력합니다.
